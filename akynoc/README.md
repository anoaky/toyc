# Usage
```sh
Akyno Compiler

Usage: akynoc [OPTIONS] <FILE>

Arguments:
  <FILE>  The source .akn file to process

Options:
  -p, --pass <PASS>  Select a specific compiler pass to test [default: all] [possible values: lexer, parser, all]
  -h, --help         Print help
  -V, --version      Print version
```
